# Resume-Ready Project Ideas

Pick projects based on the job you want, not only the tools you want to practice.

## 1. Job Application Tracker

Build a full-stack app for tracking applications, interviews, contacts, notes, and follow-ups.

Good resume signals:

- CRUD workflows with authentication.
- Database schema design.
- Search, filters, and status analytics.
- Calendar reminders or email follow-up automation.
- Tests for core business logic.

Possible stack:

- React or Next.js
- Node/Express, Django, or FastAPI
- PostgreSQL or SQLite
- Playwright or Jest for tests

## 2. Personal Finance Dashboard

Build a dashboard that imports transactions, categorizes spending, and shows trends.

Good resume signals:

- Data cleaning and validation.
- Charts and summary metrics.
- CSV import/export.
- Category rules and recurring transaction detection.
- Privacy-conscious handling of local data.

Possible stack:

- Python, pandas, Streamlit, or FastAPI
- React for a richer frontend
- SQLite or DuckDB

## 3. AI Study or Research Assistant

Build a tool that stores notes, searches them semantically, and generates study questions or summaries.

Good resume signals:

- Practical AI integration.
- Document parsing and chunking.
- Retrieval and ranking.
- Prompt design with guardrails.
- Evaluation examples that show quality.

Possible stack:

- Python or TypeScript
- OpenAI API
- SQLite, Postgres, or a vector store
- A small web UI

## Recommended First Project

Start with the Job Application Tracker if your resume target is software engineering or full-stack roles. It is easy for recruiters and interviewers to understand, and it naturally creates opportunities to show frontend, backend, database, testing, and deployment skills.
